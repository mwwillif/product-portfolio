# Screenshots

Add screenshots here after removing sensitive information.

## Recommended Screenshots

### 1. Terminal Run

Show the tool running from start to finish.

Example:

```text
Loading job description...
Analyzing job requirements...
Loading resume template...
Updating approved sections...
Saving tailored DOCX...
Exporting PDF...
Complete.
```

### 2. Output Folder

Show the generated files.

Example:

```text
Mete_Williford_Resume_ExampleCompany.docx
Mete_Williford_Resume_ExampleCompany.pdf
```

### 3. Template Markers

Show a sanitized example of the resume template markers.

Example:

```text
[[SUMMARY_START]]
Sanitized summary content here.
[[SUMMARY_END]]
```

### 4. Before and After

Show a sanitized before-and-after section.

Example:

```text
Before:
Product manager with experience in data and operations.

After:
Technical product manager with experience building AI-assisted workflow automation, data products, and execution tracking systems.
```

## Do Not Include

- Real personal address.
- Private phone numbers unless intentionally public.
- API keys.
- OpenAI credentials.
- Private file paths.
- Real job application records.
- Unredacted resumes.
- Employer-confidential information.

# Mete Williford Product Portfolio

## AI Automation | Data Platforms | Workflow Systems | Financial Tools

I build practical products that turn manual, fragmented workflows into structured, automated systems. My work spans AI automation, data platforms, financial tools, public-sector contracting, mortgage lead generation, and enterprise workflow modernization.

This portfolio is organized by project. Each project has a main case study README and supporting folders for sanitized examples, diagrams, and screenshot notes.

## Featured Projects

### AI Resume Tailoring Engine

A Python-based AI resume tailoring system that analyzes job descriptions, extracts required skills and domain signals, and rewrites approved resume sections using a marker-based Word template.

[View Case Study](projects/ai-resume-tailor/README.md)

### Bay Area Mortgage Calculator and Lead Generation Platform

A customer-facing mortgage calculator website designed to capture high-intent Bay Area homebuyer leads through affordability, mortgage payment, and closing cost calculators.

[View Case Study](projects/bay-area-mortgage-calculator/README.md)

### Automated Crypto Trading System

A Python-based automated trading system that ingests market data, calculates indicators, evaluates trade signals, manages simulated and live trade logic, and logs portfolio performance.

[View Case Study](projects/crypto-trading-bot/README.md)

### Public Sector Contract Search and Triage System

A Python-based contract monitoring system used to search, parse, normalize, and track public procurement opportunities.

[View Case Study](projects/contract-searcher/README.md)

### Government Contract Form Automation Tool

A workflow tool designed to streamline repetitive government contracting form completion by reusing structured business information and reducing manual entry.

[View Case Study](projects/contract-form-filler/README.md)

### Enterprise Execution Tracking and Reporting Platform

An enterprise workflow and reporting system built using Azure DevOps, Power Automate, and Power BI to connect planning commitments with live execution data.

[View Case Study](projects/enterprise-execution-platform/README.md)
